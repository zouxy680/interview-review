# Interview Review - 面试复盘助手

面试是消耗战，但准备可以是积累战。

每次面完，把录音转写文本丢给 AI，5 分钟拿到三份整理好的文档，帮你系统性复盘、持续改进。

## 生成的三份文档

| 文档 | 内容 | 用途 |
|:-----|:-----|:-----|
| **面试总复盘.md** | 横版表格，逐条分析每个问题：原始回答、亮点、不足、改进建议、推荐答法 | 左右对照，快速定位薄弱点 |
| **面试问题总结.md** | 按环节分类的问题清单（自我介绍/项目深挖/技术/行为/职业规划等） | 面试前翻一遍，知道该重点准备什么 |
| **完整录音整理.md** | 清洗后的完整对话（修复识别错误、规范术语、补充缩写全称） | 不用重新听录音，直接看文档 |

## 安装

### 方式一：作为 Claude Code Plugin 安装（推荐）

```bash
# 1. 添加 marketplace
/plugin marketplace add zouxy680/interview-review

# 2. 安装插件
/plugin install interview-review
```

安装后即可使用 `/interview-review` 命令。

### 方式二：直接复制 Skill

将 `skills/interview-review/SKILL.md` 的内容复制粘贴到任意 AI 对话（ChatGPT、Claude、Gemini 等）中，然后发送你的转写文本即可。

## 使用

```bash
# 传入转写文本文件路径
/interview-review ./transcript.txt

# 或直接粘贴文本
/interview-review 这是面试的转写文本内容...
```

## 工作流程

```
录音 → 语音转文字(飞书/通义/讯飞) → 丢给 skill → 3 份文档到手
```

每次面试会自动创建独立文件夹（如 `2026-04-07_字节跳动_前端`），积累下来就是你的私人面试题库。

## 为什么这样做

- 面试里很多问题是高频出现的，第一次答不好，不去整理，第二次还是原样
- 四五十分钟的录音没勇气重新听，拖几天细节全忘
- 每次花 5 分钟整理，积累的文档下次还能用
